This Admin console monitors and dynamically ensures the Docker Micro-Service Architecture remains available and scales to meet demand.

This dashboard page shows the overall status of the service and shows any service events.

The performance page shows performance data about the service.

The containers page shows the list of Docker containers running on the system.

The console page lets you control the Auto-Scaler, Chaos-Monkey and Zero-Downtime upgrade functions.
